https://skysiss.com реализует подобие приложений через IM сервисы: 
skype, jabber (xmpp протокол, например, @gmail.com) и whatsapp

на данный момент доступны следующие приложения:
BASE app
    Статистика пользователя
    Управление приложениями
    Выбор языка (en, ru, es, de)

DEV app
    Хеш генератор (генерация из введенной строки всевозможных хешей)
    Хабрахабр Помощник (генерирует ссылки на лучшие за сутки статьи с коротким описанием)
    Мониторинг Linux сервера (возможность смотреть нагрузку и статистику по linux серверам)

OFFICE app
    Калькулятор
    Курсы валют (включая нац банки СНГ и крупные международные + bitcoin)
    Мировое время (смещения, таймзона, восход и закат)
    Мировые новости (новостные фиды мировых информагенств на оригинальных языках (en, ru, es, de), по аналогии с habr)

LIFESTYLE app
    xMessenger (возможность отправки сообщенеий из любого в любой из трех доступных IM)
    Погода (погода по введеному городу)
    Гороскоп
    Волшебный шар 8ка (аналог известной игрушки)
    Игра "Башня Ханоя" (через ascii art)

Имеется API для сборки информации (кроме внутренних целей, используется в DEV app - Мониторинг Linux сервера),
то есть добавялть можно любую динамическую информацию через public API. 
Статику можно просто добавить в бд. Или отдельным приложением.
